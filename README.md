# Inventory System for Chemicals

## Summary ##
Chemical Inventory Management System (CIMS) is a Python based application developed for laboratory chemical management. It is useful for classifying chemicals, searching for chemicals, tracking inventory, and storing data in CSV format.

## Features
- Laboratory chemicals classification
- Search chemicals by name
- add new chemicals
- CSV format store inventory
- Show inventory statistics- Recommend storage methods based on chemical hazards.

## Technologies Used
- Python 
- CSV format
- File handeling
- Dictionaries 
- Functions
